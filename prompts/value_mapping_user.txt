para unificarlas según las reglas indicadas.

INSTRUCCIONES

- No crees valores nuevos.

- Mapea siempre los valores de no respuesta a null.

FORMATO DE SALIDA

VALUE_MAPPING = {
    "Nombre de la variable": {

        "No sabe": null
    }
